# Drag & Drop todo list with VueJS

Used stack: 
 - VueJS
 - Typescript + Decorators 
 - Babel
 - HTML Drag & Drop API 

Further information can be found on [medium]().

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
